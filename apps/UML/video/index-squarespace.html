Counting Tips

  1. Only count fish that swim completely upstream (across the left edge of the video), including any you might see at the start of the video
  2. Ignore any fish that swim back downstream (to the right)
  3. If you don't see any fish, submit a count of zero
  4. Use pause, slow down, or full screen if there are too many fish to count
